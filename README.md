# SAO-to-Minibadge-universal-adapter
This is a minibadge format adapter that can be soldered for SAO to Minibadge or Minibadge to SAO usage. This will allow Minibadges to be used on DEFCON badges or SAOs used on Minibadge displays/badges.

All the PCB KiCad files are provided with the gerber zip so you can just buy the boards with that.

Parts are simple with just 16 male header pins or 2 female single row 8 hole adapter and then either a male or female SAO adapter depending on which way you are going. See video below for details.
